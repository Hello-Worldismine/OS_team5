# Operating-System-TEAM-5

### build

<pre>make</pre>

### clean

<pre>make clean</pre>

### fclean

<pre>make fclean</pre>

### re

<pre>make re</pre>

### run

<pre>./TEAM5_LINUX</pre>

### How to handling segmentation fault

<pre>gcc [all source files] -fsanitize=address -g3 -o [output]</pre>

### If you want to build faster

<pre>make -j</pre>
